# YouTube Video Etiket Bulucu
<strong>Kurulum ve veritabanı gerekmez.</strong> Direkt localhost'a atıp çalıştırabilirsiniz.
# Giriş Ekranı
![PHP Youtube Video Etiket Bulucu](https://image.prntscr.com/image/3-UxRGCuRzGoOUyMYrjoLA.png)
• Sadece <strong>YouTube URL</strong>'lerini kabul ediyor. Eğer link hatalı veya YouTube linki değil ise <strong>işlem yapmaz</strong>.
# Sonuc Ekranı
![PHP Youtube Video Etiket Bulucu](https://image.prntscr.com/image/ZirGjGMpTbunmkGG1vDNqA.png)
• <strong>Etiketler</strong> kısmı kaydırılabilir. İstenilen bir <strong>etiketin üzerine tıklayınca</strong> otomatik olarak kopyalanmış oluyor.<br>
• Videonun en yüksek kalitedeki <strong>thumbnaili</strong> gösteriliyor. <strong>İndir butonu</strong> ile indirilebilir.<br>
• En altta ise etiketlerin YouTube için özel olarak <strong>(,)</strong> ile ayrılmış halini gösteriyor. <strong>Kopyala butonu</strong> ile kopyalanıp kullanılabilir.
# Uyarı
<strong>images</strong> klasörü silinmemelidir. Thumbnail indirme işlemi images klasörüne kaydedildikten sonra gerçekleşiyor.
